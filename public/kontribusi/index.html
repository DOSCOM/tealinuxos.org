Konten Panduan

Dokumentasi panduan TeaLinuxOS ditulis dengan format Markdown dan di-host di github. Kontribusi baik berupa perbaikan kesalahan penulisan, pembaruan, bahkan translasi akan mebuat lebih manis.

Dokumentasi disusun oleh Doscom dengan dua bahasa, yaitu Bahasa Indonesia pada url http://tealinuxos.org/panduan/ dan Bahasa Inggris pada url http://tealinuxos.org/en/panduan/.

Pada bagian bawah setiap halaman panduan tersedia link yang menuju file markdown di github yang bisa diperbarui langsung melalui github.
